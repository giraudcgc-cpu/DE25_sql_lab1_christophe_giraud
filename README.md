# DE25_sql_lab1_christophe_giraud

The purpose of this lab:
To use the knowledge of SQL and relational database learnt from this course to find insights from DVD rental data. You are a data analyst in a DVD rental company. There is some legacy data stored in sqlite database, your manager "The Manager" asks you to migrate it to duckdb, analyze them and present the insights to The Manager. Create a public github repo to save your work over the entire project.
